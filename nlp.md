# 3. NLP的过程
   NLP的核心过程就是： 文本 → 数字 → 模型处理 → 输出结果
  
   我喜欢自然语言处理 →  ["我", "喜欢", "自然语言处理"]  →  [12, 305, 879]
   标准过程是：
   原始文本 → 文本清洗 → 分词 / Tokenization → 文本表示 → 模型处理 → 输出结果
### 3.1 原始文本
  原始文本可能长这样：
  <strong> 这个手机真的很好用！！！物流也很快，五星好评。</strong>  它里面可能包含：标点，表情，错别字，空格，HTML 标签，特殊符号，噪声文本。
### 3.2 文本清洗
  <strong> 这个手机真的很好用 物流也很快 五星好评 </strong> 可能包括：去除多余空格，统一大小写，去除 HTML 标签 ，处理表情符号，处理特殊字符，繁简转换，拼写纠错。
### 3.3 分词 / Tokenization
   也叫做：分词 / 切词  我喜欢自然语言处理 →  ["我", "喜欢", "自然语言处理"]
### 3.3 文本表示
   分词之后，还需要把 token 变成数字。   ["我", "喜欢", "自然语言处理"] → [1, 2, 3]
   这只是最粗暴的文本表示办法：还有一些更好的方法：后来有了更好的文本表示方法，比如： One-hot， Bag of Word， TF-IDF， Word2Vec， Embedding， Transformer。
### 3.4 模型处理
   文本转成数字后，就可以交给模型处理。模型可以是：朴素贝叶斯 ，逻辑回归， SVM， RNN， CNN， Transformer， LLM。
